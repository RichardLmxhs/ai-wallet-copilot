# AI Wallet Copilot

智能钱包分析助手，能够对指定钱包地址进行资产分析、行为分析和风险评估。

## 功能

- 钱包资产分析
- 交易历史追踪
- AI驱动的风险分析
- 行为模式识别
- 钱包操作建议

## 技术栈

- Backend: Golang
- Database: PostgreSQL, Redis
- Blockchain: Ethereum (支持其他EVM兼容链)
- AI: Large Language Model Integration

## 快速开始

```bash
# 使用Docker Compose启动
cd deployments/docker
docker-compose up -d
```
